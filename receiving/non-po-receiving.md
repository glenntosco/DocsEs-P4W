# Recepción Sin OC

La recepción sin OC es para recibir artículos adicionales enviados por el proveedor que no estaban en la Orden de Compra original. Aún requiere un documento de compra base.

{% hint style="info" %}
Esta función es para recibir artículos adicionales en una Orden de Compra **EXISTENTE**. No es para recibir artículos sin ninguna Orden de Compra.

Para recibir un artículo que no tiene una Orden de Compra del proveedor, deberás usar la función de ajuste de inventario.
{% endhint %}
