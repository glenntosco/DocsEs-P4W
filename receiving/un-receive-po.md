# Des-recibir OC

En caso de que un producto sea recibido incorrectamente y la Orden de Compra no haya sido cerrada, existe la posibilidad de revertir la recepción del (los) producto(s).

{% hint style="danger" %}
Recuerda que el proceso de recepción es, en términos absolutos, el proceso más importante de un almacén. Las malas prácticas de recepción causarán muchos retrasos en otros procesos del almacén.
{% endhint %}
