# Información General de Recepción

* **Recepción** — El ícono de Recepción contiene todas las funciones para la recepción de productos.
  * **Lista de OC** — Para almacenes de tipo e-commerce, permite seleccionar de una lista de órdenes de compra abiertas.
  * **Recepción de OC** — Permite recibir Órdenes de Compra.
  * **Foto de OC** — Para documentar productos dañados durante la recepción.
  * **Código de Barras de Producto** — Para imprimir códigos de barras faltantes en los productos.
  * **Des-recibir Orden de Compra** — Permite revertir la recepción de un producto si fue ingresado incorrectamente. (La Orden de Compra debe estar en estado Abierto para este proceso.)
  * **Recepción Sin OC** — Permite al usuario recibir un producto que no está en la Orden de Compra específica. (Debe existir una orden de compra para el proveedor; esta función no es para recibir de forma aleatoria cualquier cosa que llegue al almacén.)
