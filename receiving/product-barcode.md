# Código de Barras de Producto

Esta función permite al receptor imprimir etiquetas de código de barras para productos que lleguen sin etiquetas, antes de recibirlos en el sistema.

{% hint style="success" %}
Dependiendo de la configuración, el sistema puede solicitar al receptor que imprima etiquetas después de recibir el producto. Estas etiquetas incluirán información clave en formato de código de barras, como número de serie, número de lote y fecha de vencimiento. Con frecuencia, los productos no vienen del fabricante con lotes y fechas en formato de código de barras.
{% endhint %}

{% hint style="danger" %}
Es fundamental que todos los productos tengan una etiqueta de código de barras antes de finalizar el proceso de recepción. Los productos sin código de barras serán muy difíciles (básicamente imposibles) de gestionar dentro del almacén, causando retrasos innecesarios en otros procesos como la recolección.
{% endhint %}
