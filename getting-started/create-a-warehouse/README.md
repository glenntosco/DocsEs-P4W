# Crear un Almacén

## Configuración del Almacén

{% hint style="info" %}
Recuerda que si P4 Warehouse está integrado a un ERP, los códigos de almacén deben coincidir con los del ERP. Consulta con tu Partner de P4 Warehouse antes de configurar los almacenes.
{% endhint %}

Configuración > Sistema > Almacén

Haz clic en el botón :new: para crear un nuevo almacén. Haz clic en el enlace del código para editar un almacén existente.

Para agregar un nuevo almacén, ingresa el código de almacén, el nombre del almacén y, si es necesario, la descripción.

Con el almacén creado, haz clic en el enlace del código de almacén para agregar información [detallada](edit-warehouse.md) del almacén.
