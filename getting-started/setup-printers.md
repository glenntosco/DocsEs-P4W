# Configurar Impresoras

Las impresoras son configuradas por tu Partner de P4 Warehouse. Por favor, contacta al partner que te vendió la solución de P4 Warehouse para obtener asistencia en la configuración de tus impresoras dentro del sistema.

#### Requisitos:

* [ ] Una computadora dedicada con Windows para ejecutar el servidor de impresión y el software de interfaz. (Debe ser un servidor existente que siempre esté encendido)
* [ ] Las impresoras deben estar configuradas en la computadora mencionada anteriormente.
  * [ ] El tipo de conexión puede ser USB o Red.
* [ ] Asegúrate de asignar nombres descriptivos a las impresoras, como: Zona123, Zona33...

{% hint style="info" %}
P4 Warehouse puede imprimir en 203 dpi, 300 dpi y 600 dpi. Asegúrate de informarle a tu Partner de P4 Warehouse el dpi de tus impresoras.
{% endhint %}

{% hint style="success" %}
Para mejores resultados al configurar el servidor de impresión, por favor contacta a tu Partner de P4 Warehouse.
{% endhint %}

Para usuarios avanzados, aquí está el enlace al instalador: [https://app.p4warehouse.com/binaries/agent.exe](https://app.p4warehouse.com/binaries/agent.exe)

Al hacer clic en el ícono de llave, se mostrará la lista de los diferentes tipos de etiquetas. Selecciona los tipos de etiquetas que deben imprimirse en esta impresora. Haz clic en Guardar al final de la pantalla después de realizar tus selecciones.
