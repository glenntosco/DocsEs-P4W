# Crear Cliente

La página de Clientes muestra un listado de todos los clientes disponibles en el sistema WMS. Incluye columnas para Número de Cliente, Nombre de la Empresa, Descripción, Persona de Contacto, Correo Electrónico, Cliente y Fecha de Creación. También hay un botón "Nuevo" en la parte superior derecha (para crear nuevos perfiles de cliente) y un botón "Actualizar" (para refrescar la lista después de cualquier cambio).

Selecciona el botón "Nuevo" para crear un perfil de cliente. Aparecerá una ventana emergente con cuatro opciones: Cliente, Número de Cliente, Nombre de la Empresa y Descripción. El campo Cliente tiene un menú desplegable para elegir un cliente de la lista disponible. Completa cada campo (la Descripción es opcional) y selecciona el botón "Guardar".

{% hint style="info" %}
El campo Cliente no se usa a menos que el sistema esté configurado para 3PL o Multi-Empresa.
{% endhint %}

La página mostrará el perfil del Cliente. En el lado izquierdo hay una sección de Detalles para ingresar correo electrónico, persona de contacto e información adicional en el campo Descripción. En la esquina superior derecha hay dos botones: "Actualizar" (guarda los cambios realizados en el perfil) y "Refrescar" (muestra los cambios aplicados). A la derecha está la sección de Direcciones para ingresar una o más direcciones del cliente, con un botón "Agregar" para nuevas direcciones. En la parte inferior se encuentra la sección de Configuración para la Tolerancia de Vencimiento de Productos (el cliente no desea recibir artículos que vencen antes del tiempo indicado; por defecto son 70 días).

Las direcciones se muestran con columnas de información para: Predeterminada, Nombre, Ruta, A/Att., Teléfono, Dirección, Ciudad, Provincia, Código Postal y Acción. Para eliminar una dirección, selecciona el ícono (X negra en caja blanca) bajo la columna Acción de la dirección correspondiente. Para editar una dirección, elige el texto del enlace bajo la columna Nombre.
