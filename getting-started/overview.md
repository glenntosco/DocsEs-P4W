# Descripción General

## Tipos de usuarios

En P4 Warehouse existen dos estilos de usuarios completamente distintos.

1. **Despachador Web** — el despachador web utiliza una computadora y un navegador web moderno para gestionar el flujo de entrada y salida del almacén. El despachador web también controla el flujo de trabajo de los usuarios PDT. Piensa en el despachador web como la torre de control de un aeropuerto.
2. **Usuario PDT** — el usuario PDT es la persona que está físicamente en el almacén ejecutando las instrucciones del despachador web. Estas son las personas que reciben físicamente las órdenes de compra y devoluciones (RMA), recolectan y despachan pedidos, y realizan conteos cíclicos de inventario. Estos usuarios trabajan desde un dispositivo móvil, generalmente con un escáner de código de barras integrado.

Se requiere experiencia en almacenes y/o en sistemas WMS para utilizar este documento. Contacta a tu partner de ventas de P4 Warehouse si encuentras temas que no comprendes completamente.

## Android

Android Nugat y superior es compatible con el Sistema P4 Warehouse. La aplicación P4W para dispositivos portátiles puede descargarse en un dispositivo Android desde [https://play.google.com/store/apps/details?id=com.pro4soft.p4w&hl=es&gl=US](https://play.google.com/store/apps/details?id=com.pro4soft.p4w&hl=es&gl=US)

Se recomienda que el dispositivo Android tenga un escáner de código de barras integrado. Los equipos de Zebra Technologies funcionan mejor, ya que P4 Warehouse está validado por Zebra para funcionar correctamente en sus computadoras móviles Android.
