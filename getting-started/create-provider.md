# Crear Proveedor

La página de Proveedores muestra un listado de todos los proveedores disponibles en el sistema WMS. Incluye columnas para Número de Proveedor, Nombre de la Empresa, Descripción, Correo Electrónico, Cliente, Fecha de Creación y Acción. También hay dos botones en la parte superior derecha: "Nuevo" (para agregar un nuevo proveedor) y "Actualizar" (para refrescar la lista después de cualquier cambio). Usa la columna Acción para eliminar el proveedor correspondiente.

Para agregar un nuevo proveedor al WMS, selecciona el botón "Nuevo" para activar un menú emergente con cuatro opciones: Cliente, Número de Proveedor, Nombre de la Empresa y Descripción. Completa cada campo y luego selecciona el botón "Guardar".

Una vez creado, selecciona el proveedor de la lista (haz clic en el enlace del Número de Proveedor) y aparecerá la página de perfil de ese proveedor. En esta página se pueden completar o editar todos los detalles del proveedor, incluyendo nombre, correo electrónico y dirección. Selecciona el botón "Actualizar" en la parte superior derecha para guardar los cambios realizados en el perfil del proveedor.
